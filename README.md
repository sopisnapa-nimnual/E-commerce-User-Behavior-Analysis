# E-commerce User Behavior Analysis

## Overview
This project explores user behavior data from an e-commerce platform.  
The goal is to understand how users interact with the platform and identify patterns in user activity.

The analysis focuses on exploring user events such as product views, cart actions, and purchases.

## Dataset
The dataset contains user activity logs from an e-commerce platform.  
Each record represents a user event such as viewing a product, adding to cart, or purchasing.

Example variables in the dataset include:
- event_time
- event_type
- product_id
- category_code
- user_id

## Analysis Performed
The notebook includes several exploratory analyses:
- Data preprocessing
- Event distribution analysis
- User activity analysis
- User activity by hour
- Visualization of user behavior patterns
These analyses help reveal how users interact with the platform and when activity levels are highest.

## Tools Used
- Python
- Google Colab
- Excel

## Key Observation
Preliminary analysis shows that user activity varies across different hours, and most events are product views compared to cart or purchase actions.
